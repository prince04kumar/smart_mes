┌─────────────────────────────────────────────────────────────┐
│  🐛 TROUBLESHOOTING                                         │
└─────────────────────────────────────────────────────────────┘

Problem: "Failed to fetch"
Solution:
  1. Check backend is running on EC2
  2. Verify EC2 Security Group allows port 5000
  3. Test: curl http://100.26.22.38:5000/health

Problem: CORS Error
Solution:
  1. Backend app.py already has CORS configured
  2. Restart backend on EC2
  3. Clear browser cache (Ctrl+Shift+Delete)

Problem: 401 Unauthorized
Solution:
  1. Logout and login again
  2. Check JWT token in localStorage (F12 → Application)
  3. Register new account if needed

Problem: Slow Response
Solution:
  1. Use smaller images (< 2MB)
  2. Check EC2 CPU usage
  3. AWS Textract processing may take time
